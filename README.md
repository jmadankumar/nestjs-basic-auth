## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### JDL Studio

- import jdl file from `docs/diagram` folder to [JDL Studio](https://start.jhipster.tech/jdl-studio/) to edit the diagram.

### Database migration

Migration scripts are available in `src/migration` directory

#### Create migration script

```
npx typeorm migration:create -n <filename>
```

#### Check list of migration which are executed and pending

```
npm run migration:show
```

#### Run migration

```
npm run migration:run
```

#### Revert migration

```
npm run migration:revert
```
